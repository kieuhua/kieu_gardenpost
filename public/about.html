About Kieu Garden Posts

Main features

It is node express application, it allows users to create posts with title, description, publishdate and upload a garden picture.

Search feature

Users can search with combination of critiria partial title, after or before particular publishdate.

Interesting feature

By move your mouse over the picture, user will see an adjacent zoom-in picture.


About Libraries

This application is designed and implemented by Kieu Hua. The following are some of the libraries are used: /p>

Mongodb and Mongoose

Mongodb is backend database to store posts with mongoose, ORM(Object-Relational Mapping)

file-pond

She use file-pond library to serialize the picture image into string in order to store it in mongodb.

method-override

She uses method-override library, so she can use PUT and DELETE.

slugify

She uses slugify library to avoid using id on url.

express-ejs-layouts

She uses express-ejs-layouts partials to organize views.
